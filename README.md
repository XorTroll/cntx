# cntx

> Pure Rust library to explore various Nintendo Switch file formats

## Usage

```toml
[dependencies]
cntx = { git = "https://github.com/XorTroll/cntx" }
```

## Supported formats

- NCA (only NCA3 version)

- PFS0

- RomFs
